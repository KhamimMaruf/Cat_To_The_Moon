# Cat To The Moon

## Cara Main
- Gunakan **A / D** untuk bergerak kiri/kanan
- Tekan **Space** untuk lompat
- Ambil ikan untuk menambah score
- Ambil **Golden Fish** untuk speed boost
- Ambil **Flying Fish** untuk jump boost

## Tujuan
Mengumpulkan score sebanyak-banyaknya dan jangan jatuh ke jurang
